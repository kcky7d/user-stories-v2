What is a user story?

A user story is a short, simple statement from a user to express a needed requirement.
It is quite simply a way for you to request a feature you'd like to see in a new tool!


Why should I write a user story?

A user story is the best way for you to express your needs.
By using this tool, you will be able to express what new features you need to do your job better. You are cutting out the middle man!


How do I write a user story?

A user story has a very simple format:
As a (insert work role), I need (insert feature request), so that (insert justification).

It's as simple as that! Here is a completed example:
As a payroll specialist, I need a tool to sum up employee salaries by department, so that I can quickly complete TPS reports.
